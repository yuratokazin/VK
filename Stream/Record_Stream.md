# How to Record a Stream

---

Как записать стрим который я провожу здесь https://live.vkvideo.ru/gitcube

---

Записать стрим с **vkvideo.ru** можно несколькими способами — ниже приведены проверенные методы с пошаговой инструкцией.

### Способ 1. Встроенная запись в OBS Studio (рекомендуемый)

**Почему лучше:** высокое качество, гибкость настроек, отсутствие водяных знаков.

1. **Установите OBS Studio**
   Скачайте с официального сайта *obsproject.com* (бесплатно, Windows/macOS/Linux).
2. **Настройте запись**
   - Откройте *Настройки → Вывод*.
   - Выберите режим: *Расширенный*.
   - Во вкладке *Запись*:
     - Путь записи: укажите папку для сохранения.
     - Формат: *MP4* или *MKV*.
     - Кодировщик: *x264* (для CPU) или аппаратный (NVENC/AMD).
     - Битрейт: 5 000–10 000 Kbps для 1080p.
3. **Добавьте источник**
   - В панели *Источники* нажмите **+**.
   - Выберите *Захват окна* или *Захват экрана*.
   - Для браузера: укажите окно с трансляцией.
4. **Начните запись**
   - Нажмите кнопку *Начать запись* в нижней панели OBS.
   - После завершения — *Остановить запись*.

### Способ 2. Браузерные расширения

**Подходит для быстрого захвата без установки ПО.**

1. Установите расширение:
   - **Chrome**: *Screencastify*, *Loom*.
   - **Firefox**: *RecordScreen*.
2. Откройте стрим на *vkvideo.ru*.
3. Запустите расширение, выберите область захвата (вкладка/окно).
4. Нажмите *Запись* → после завершения сохраните файл.

**Минусы:** может быть водяной знак (в бесплатных версиях), ограничение по времени.

### Способ 3. Встроенные инструменты ОС

**Windows:**

1. Нажмите `Win + G` → откроется игровая панель.
2. Кликните *Начать запись* (или `Win + Alt + R`).
3. Для остановки — снова `Win + G` → *Стоп*.
4. Видео сохранится в *Видео → Captures*.

**macOS:**

1. Откройте *QuickTime Player*.
2. Выберите *Файл → Новая запись экрана*.
3. Нажмите красную кнопку → выберите область записи.
4. Для остановки — значок остановки в меню-баре.

### Способ 4. Онлайн‑сервисы

**Примеры:** *Clipchamp*, *Screencapture.com*.

1. Перейдите на сайт сервиса.
2. Выберите *Запись экрана* → разрешите доступ к экрану.
3. Укажите область захвата (вкладка браузера).
4. Сохраните результат после записи.

**Важно:**

- Проверьте скорость интернета (минимум 5 Мбит/с для 720p).
- Закройте лишние приложения, чтобы избежать лагов.
- Для OBS: в *Настройки → Видео* установите разрешение, совпадающее с трансляцией.

### Что выбрать?

- **Для качества и контроля** → OBS Studio.
- **Для быстрого захвата** → браузерные расширения или встроенные инструменты ОС.
- **Без установки ПО** → онлайн‑сервисы.
